Project Title

Attendance Tracking and Payroll Calculation Mobile App

Project Description

Developed a mobile application for employee attendance management and payroll calculation using Flutter and Dart. The application enables organizations to monitor employee attendance, manage work records, automate salary calculations, and track employee activities through a responsive and user-friendly mobile interface.

Key Features
Employee login and authentication
Daily attendance check-in/check-out
Real-time attendance tracking
Automatic working hours calculation
Lunch break and break time management
Overtime calculation
Leave and absence management
Payroll and salary calculation
Monthly payroll and attendance reports
Responsive mobile UI design
Admin dashboard and employee management
Employee profile and attendance history
Secure cloud data storage
Technologies Used
Frontend/Mobile App: Dart, Flutter
Backend & Database: Firebase / MongoDB
Authentication: Firebase Authentication
Tools: Git, GitHub
